    Mengenlehre

    Eine Menge ist eine Sammlung von Objekten. Das klingt erst mal sehr banal, bildet aber den Grundbaustein der Mathematik.

    Im Folgenden werden wir kennenlernen, was es für Mengen gibt, wie wir diese aufschreiben können und warum sie für uns nützlich sind.

    Grundlegende Zahlenmengen

    Video

    Das \(\in\)-Zeichen

    Um anzuzeigen, dass ein Element in einer Menge enthalten ist, nutzen wir das \(\in\)-Zeichen. Wenn ein Element nicht in einer Menge enthalten ist, nutzen wir das \(\notin\)-Zeichen

    Beispiel

    \[ 5\in \mathbb{N}, \]
    \[ \frac{5}{3}\notin \mathbb{N} \]

    Prüfung der Zugehörigkeit

    Füllen Sie die folgenden Lücken mit \(\in\)- oder \(\notin\)-Zeichen.

    a) \(7 \text{ ___ } \mathbb{N}\)

    b) \(-3 \text{ ___ } \mathbb{N}\)

    c) \(0 \text{ ___ } \mathbb{Z}\)

    d) \(\frac{1}{3} \text{ ___ } \mathbb{Q}\)

    e) \(\sqrt{2} \text{ ___ } \mathbb{Q}\)

    f) \(5.5 \text{ ___ } \mathbb{Z}\)

    g) \(3.14159 \text{ ___ } \mathbb{R}\)

    Lösung

    a) \(7 \in \mathbb{N}\) (\(7\) ist eine natürliche Zahl)

    b) \(-3 \notin \mathbb{N}\) (\(-3\) ist keine natürliche Zahl)

    c) \(0 \in \mathbb{Z}\) (\(0\) ist eine ganze Zahl)

    d) \(\frac{1}{3} \in \mathbb{Q}\) (\(\frac{1}{3}\) ist eine rationale Zahl)

    e) \(\sqrt{2} \notin \mathbb{Q}\) (\(\sqrt{2}\) ist keine rationale Zahl)

    f) \(5.5 \notin \mathbb{Z}\) (\(5.5\) ist keine ganze Zahl)

    g) \(3.14159 \in \mathbb{R}\) (\(3.14159\) ist eine reelle Zahl)
